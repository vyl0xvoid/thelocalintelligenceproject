The Local Intelligence Project: Our Four Pillars

A brief overview of the initiatives we will build through local branches, along with the first steps for each.

OpenRoot

Roadmap: build a plain-language AI literacy track that branches can use for learning circles, public explainers, local workshops, and shared community guides.

First steps: collect the most common beginner questions, write the first starter guide, host one introductory call, and ask members to test the language with people in their own circles.

Tinbot Exchange

Roadmap: create a trusted exchange for reclaimed laptops, routers, parts, tools, and small compute so useful hardware can be repaired, traded, documented, and reused.

First steps: define donation and safety rules, make a drive-wipe checklist, create a simple inventory template, and start a small list of available parts and local hardware needs.

Myceli-Net

Roadmap: test small offline communication systems and local knowledge-sharing practices that can help branches stay coordinated without depending fully on centralized platforms.

First steps: document a beginner mesh setup, collect privacy and safety notes, run a small local range test, and turn the results into a repeatable starter guide.

Solar Link

Roadmap: develop the physical branch model: repair benches, gardens, local servers, workshops, gathering space, and practical infrastructure that connects technology back to place.

First steps: define what counts as a Solar Link, identify possible host spaces, list basic equipment needs, and document one small pilot setup that other branches can adapt.
